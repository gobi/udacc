# UDA Cycling Club

Унадаг дугуйн клубын гишүүдэд зориулсан вэб систем.

## Боломжууд

### Аялал
- Шинэ аялал зохион байгуулах
- GPX файлаар маршрут оруулах (RideWithGPS, Strava-с татах)
- Газрын зураг дээр маршрут, эхлэх/дуусах цэг, даваанууд харагдана
- Уулзах газар тэмдэглэх
- Өндрийн профайл, нийт км, өндөрт авирсан метр автоматаар тооцоолно
- Аялалд бүртгүүлэх, цуцлах
- Ирц бүртгэх, явсан км оруулах

### Гишүүд
- Бүртгүүлэх (имэйл эсвэл Facebook-р)
- Хувийн профайл - нийт явсан км, аялалын тоо
- Тэргүүлэгчдийн жагсаалт (хамгийн их км явсан)

## Технологи

- **Backend:** Go, Fiber, PostgreSQL
- **Frontend:** Next.js, React, TailwindCSS, Leaflet
